# UBC Knights Stats

A stats dashboard for the UBC Knights hockey team. Black and gold theme with dark/light mode, sortable leaderboards, goalie stats, team analytics, and season history.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## CSV Formats

**Skaters** (`skaters.csv`):
```csv
Player,GP,G,A,P,PM
John Smith,16,24,9,33,12
```
Column names are flexible — handles `Goals`/`G`, `Assists`/`A`, `Points`/`P`/`PTS`, `PIM`/`PM`, etc. Points auto-calculate from `G + A` if omitted. PPG is calculated automatically.

**Goalies** (`goalies.csv`):
```csv
Player,GP,GAA,SV%,SV,GA,SA,MIN,OTL
Jane Doe,20,6.84,0.844,494,91,585,798,2
```

**Games** (`games.csv`):
```csv
Date,Opponent,GF,GA,Result,OT,Home
2025-10-15,Whalers,4,2,W,0,1
```

**Playoff CSVs** use the same format as regular season, prefixed with `playoffs-`:
- `playoffs-skaters.csv`, `playoffs-goalies.csv`, `playoffs-games.csv`
- Only add these for seasons that had playoffs

**Awards** (`awards.json`):
```json
[
  { "name": "Empty Net Assassin", "description": "Most empty net goals", "winner": "John Smith", "stat": "2 EN goals" },
  { "name": "Glue Guy(s)", "description": "Getting the most spares", "winners": ["Jane Doe", "Bob Jones"] }
]
```
Use `"winner"` for a single winner or `"winners"` (array) for multiple. Only include manual awards — auto awards (Ironman, Cheechoo Train, etc.) are computed from the data.

**Recaps** (`recaps.json` / `playoffs-recaps.json`):
```json
[
  {
    "date": "2024-09-16",
    "opponent": "Smoking Aces",
    "gf": 7, "ga": 3,
    "result": "W", "ot": false,
    "periods": [
      { "name": "1st", "shotsHome": 12, "shotsAway": 8, "goalsHome": 2, "goalsAway": 0 }
    ],
    "goals": [
      { "period": "1st", "time": "00:03:12", "team": "Knights", "scorer": "John Smith", "assists": ["Jane Doe"] }
    ],
    "penalties": [
      { "period": "1st", "time": "00:06:11", "team": "Knights", "player": "Bob Jones", "type": "Interference", "severity": "Minor", "minutes": 2 }
    ]
  }
]
```
For the current season, recaps are generated by `scripts/ingestion.py` (SportNinja API). For older seasons (2023-24, 2024-25), recaps were scraped from Pointstreak using `scripts/pointstreak_scraper.py`.

**All Time** stats are auto-aggregated from all season files — no separate file needed.

## Features

### Tabs
- **Skaters** — sortable leaderboard with GP, G, A, PTS, PPG, PIM. Click any player row to expand a season-by-season breakdown card. Leader cards for top scorer, goal leader, assist leader.
- **Goalies** — sortable table with GAA, SV%, SV, GA, SA, MIN, OTL. SV% semicircle gauges for each goalie.
- **Team** — team record (W-L-OTL), goals for/against, goal differential, roster size, PIM, team SV%. Current form indicator (last 5 games). Opponent breakdown table (sortable). Recent results with expandable game timelines (period-by-period scoring, goals with assists, penalties). Goal distribution donut chart. Comeback wins section.
- **Awards** — end-of-season awards displayed as a card grid (regular season only). Auto awards are computed from stats data (Ironman, Cheechoo Train, Adam Banks, Jumbo Joe, Frequent Flyer, Ghost Checker, Heartbeat Hero). Manual awards are loaded from `awards.json`.

### Season Navigation
- Current season tab
- All Time (aggregated across all seasons)
- History dropdown for past seasons
- Regular Season / Playoffs toggle (appears when playoff data exists for the viewed season)

### Visualizations
- Leader cards (top scorer, goal leader, assist leader)
- Points breakdown bar chart (goals vs assists per player)
- Goal distribution donut chart (team tab)
- SV% semicircle gauges (goalies)
- Season timeline (colored dots by month: green=W, red=L, gold=OTL)
- Cumulative points chart (top 5 scorers progression)
- Milestone tracker (players within 5 of a career milestone)
- Current form indicator (W/L/OTL boxes)

### Other
- Dark/light mode toggle
- Spotify playlist link in header
- Expandable player cards with per-season sparklines
- Sortable tables throughout

## Pull Stats from SportNinja

```bash
# Install Python dependencies
poetry install
# Or: pip install requests python-dotenv pyyaml

# Run ingestion
python -m scripts.ingestion
```

This pulls skater stats, goalie stats, and game results from the SportNinja API and writes CSVs to `public/seasons/{season}/`.

**First-time setup:**
1. Copy `config/.env.example` to `config/.env` (or create it)
2. Open your team page on [app.sportninja.com](https://app.sportninja.com)
3. Open DevTools (F12) > Network tab > filter by Fetch/XHR
4. Find any stats request > copy the Bearer token from the Authorization header
5. Set `SPORTNINJA_TOKEN=<your token>` in `config/.env`

API configuration is in `config/ingestion.yml`:
```yaml
schedule_id: "oLtOPiBmSkBVNlIo"
team_id: "vzPOLIu6nhIQ8GXh"
season: "2025-26"
api_base: "https://metal-api.sportninja.net/v1"
```

**Update workflow:** grab token > `python -m scripts.ingestion` > `git push` > auto-deploys.

## Pointstreak Scraper (Older Seasons)

A one-time script that scrapes game box scores from Pointstreak for the 2023-24 and 2024-25 seasons:

```bash
python scripts/pointstreak_scraper.py
```

Game IDs are hardcoded in the script (sourced from Pointstreak team schedule pages). The scraper parses each box score page and outputs `recaps.json` and `playoffs-recaps.json` files matching the same format used by the SportNinja ingestion. No API key required — data is scraped from public pages.

## Adding a New Season

1. Create the directory: `public/seasons/{season-id}/`
2. Add `skaters.csv`, `goalies.csv`, and optionally `games.csv`
3. Optionally add `recaps.json` for game timelines (expandable game details with goals, penalties, period summaries)
4. Optionally add `playoffs-skaters.csv`, `playoffs-goalies.csv`, `playoffs-games.csv`, `playoffs-recaps.json` (toggle appears automatically when these exist)
5. Optionally add `awards.json` for manual end-of-season awards (regular season only)
5. Add the season to the `SEASONS` array in `src/App.jsx`:
```js
const SEASONS = [
  { id: "2025-26", label: "2025-26", dir: "/seasons/2025-26" },
  { id: "2024-25", label: "2024-25", dir: "/seasons/2024-25" },
];
```

## Customize

In `src/App.jsx`, edit the config at the top:
```js
const CONFIG = {
  teamName: "UBC Knights",
  established: "2023",
};
```

The knight helmet logo is an SVG component (`KnightLogo`). To use your own logo, replace it with `<img src="/logo.png" />` and drop the image in `public/`.

## Deploy to Vercel

1. Push to a GitHub repo
2. Go to [vercel.com](https://vercel.com) > sign in with GitHub
3. Import your repo — framework auto-detects as Vite
4. Deploy

To update: run ingestion, commit & push. Vercel auto-redeploys.

## Tech Stack

- **Frontend:** React 18 + Vite, PapaParse for CSV parsing
- **Ingestion:** Python 3.11+, requests, PyYAML
- **Styling:** Inline CSS with dark/light theme support
- **Fonts:** Outfit, DM Mono (Google Fonts)
